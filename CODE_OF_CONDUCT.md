# 社区行为准则

见微希望建立一个友善、务实并尊重边界的开源社区。

## 我们鼓励

- 围绕事实、复现步骤和具体方案讨论问题。
- 尊重不同经验水平，解释术语和必要背景。
- 对代码提出批评时聚焦行为和影响，而不是评价个人。
- 主动保护日志、截图和示例数据中的隐私。
- 承认第三方平台能力可能变化，不把维护者当作平台客服。

## 不可接受

- 骚扰、侮辱、歧视、威胁或持续的人身攻击。
- 公开他人的个人信息、凭据、Cookie、账号状态或私密内容。
- 引导绕过平台安全机制、滥用账号或进行未经授权的数据采集。
- 在问题中投放垃圾广告、恶意链接或与项目无关的推广。
- 明知存在安全漏洞仍公开可直接利用的细节。

## 处理方式

维护者可以编辑、隐藏或删除违反准则的内容，并视情况限制参与权限。需要私下报告的安全问题请遵循 [SECURITY.md](SECURITY.md)；其他行为问题可以通过 GitHub 账号主页联系维护者，请不要在公开 Issue 中披露当事人的个人信息。

处理决定会尽量说明事实、规则和后续行动，同时保护报告者与受影响者的隐私。
